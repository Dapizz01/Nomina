# Nomina

Nomina is a client-side web application for editing MP3 and FLAC metadata directly on your file system.

## Setup
### Install dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```

### Testing
```bash
# Run unit tests (Vitest)
npm run test:unit

# Run unit tests and generate coverage report
npm run test:coverage

# Run full system E2E tests (Playwright)
npm run test:e2e
```
